## Overview

You can try it out here: [https://secure-image-converter.com](https://secure-image-converter.com)

This is a Next.js project that allows you to convert images types completly in the browser. No backend needed.

## Motivation

1. Create a secure and easy to use image converter. Allot of online image converters are shady looking or filled with ads and popups.

2. Learn about Next.js and Vercel deployments.

## Setup project

```bash
# install dependencies
npm install
# start dev server
npm run dev
# visit http://localhost:3000
```

## Tech Stack

This project uses Next.js with React and Shadcn.
